# AI HSE Management System - Frontend

A cutting-edge, futuristic Health, Safety, and Environment (HSE) management frontend built with React, Tailwind CSS, and Framer Motion. This system provides real-time safety monitoring, AI-powered risk assessment, and comprehensive incident management.

## 🚀 Features

### ✨ Modern UI/UX
- **Dark Theme Design**: Sleek, professional dark interface with glassmorphism effects
- **Responsive Layout**: Fully responsive design that works on all devices
- **Smooth Animations**: Framer Motion powered animations and transitions
- **Interactive Elements**: Hover effects, micro-interactions, and visual feedback

### 🎯 Core Functionality
- **Real-time Dashboard**: Live safety metrics, zone status, and activity monitoring
- **AI Risk Analysis**: Intelligent risk assessment with predictive analytics
- **Incident Tracking**: Comprehensive incident management with AI recommendations
- **Work Permit System**: Digital permit management with AI validation
- **AI Chat Assistant**: Intelligent safety guidance and protocol assistance

### 🔒 Safety Features
- **Real-time Alerts**: Instant notification system for safety issues
- **Risk Scoring**: AI-powered risk assessment with color-coded indicators
- **Zone Monitoring**: Real-time status tracking for different work areas
- **Compliance Tracking**: Automated safety protocol monitoring

## 🛠️ Technology Stack

- **Frontend Framework**: React 18.2.0
- **Styling**: Tailwind CSS 3.3.5
- **Animations**: Framer Motion 10.16.4
- **Icons**: Lucide React 0.263.1
- **Build Tool**: Vite 5.0.0
- **Package Manager**: npm

## 📦 Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd ai-hse-demo/frontend
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
